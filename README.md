# Web Crawler

Boot.dev challenge: building a web crawler using Go.

## Requirements

- Go 1.20 or later

## Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/Elianamos29/crawler.git
    cd crawler
    ```

2. Build the project:
    ```bash
    go build
    ```

## Usage

Run the crawler with a starting URL:
```bash
./crawler https://example.com
```
